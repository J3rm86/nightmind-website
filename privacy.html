Introduction

NightMind is committed to protecting user privacy. This Privacy Policy explains how information is handled when using the NightMind mobile application.

Privacy First: NightMind is designed with privacy at its core. Dream journal data is stored locally on the device and encrypted. NightMind does not collect, store, or have access to personal dream content.

Local Data Storage

All dream data stays on the device:

  • Dream Entries: Stored locally in an encrypted database on the device
  • Voice Recordings: Processed entirely on-device using local AI and deleted immediately after transcription
  • Settings and Preferences: Stored locally on the device
  • Generated Videos: Stored locally on the device

Data NOT Collected

  • No analytics or usage tracking
  • No behavioral data collection
  • No storage of dream content on external servers
  • No selling or sharing of personal information

Cloud Features (AI Analysis & Video Generation)

When using AI-powered features, the following data is temporarily transmitted to secure servers for processing:

  • Dream Text: Sent for AI analysis or video generation, then immediately discarded after processing
  • Device ID: An anonymous identifier used for rate limiting and credit tracking
  • RevenueCat User ID: Used to verify subscription status and manage credits

All transmissions use secure HTTPS encryption. Dream content is NOT stored on servers after processing is complete.

Microphone Permission

NightMind requests microphone access to enable voice recording of dreams. This feature is optional.

  • Audio is processed locally on-device using Whisper AI
  • Voice recordings are never uploaded to external servers
  • Audio files are automatically deleted after transcription completes

Third-Party Services

  • RevenueCat: Processes subscriptions and in-app purchases. Receives anonymous user ID for purchase management. RevenueCat Privacy Policy
  • Cloudflare Workers: Hosts the API for AI features. Receives dream text temporarily for processing only. Cloudflare Privacy Policy
  • Apple App Store / Google Play: Handles payment processing for purchases

Data Security

  • Local dream database is encrypted
  • All network communications use HTTPS encryption
  • No cloud storage of personal dream content
  • No account or login required to use the app

User Control

  • Export dream journal data at any time via the app settings
  • Delete all data from within the app
  • Use the app entirely offline (AI features require internet connection)
  • Deny microphone permission and use text entry only

Children's Privacy

NightMind is not intended for children under 13 years of age. No personal information is knowingly collected from children under 13.

Changes to This Policy

This Privacy Policy may be updated from time to time. Changes will be posted on this page with an updated "Effective Date."
